> *"每一只蚂蚁都是炸药工程师。每一个坩埚都是一次赌博。每一次爆炸都是一场物理喜剧。"*

## 这是什么

**AntChem** 是一款 2D 像素风独立游戏。你是一只行军蚁——不是普通工蚁，而是蚁巢里最危险的那只：**炸药工程师**。

你的实验室是蚁巢深处一个勉强挖出来的小洞穴。照明靠萤火虫，工作台是碎石板搭的，坩埚是个掏空的坚果壳。在这里，你从大自然中采集原料——硫磺石、硝石晶、碳粉、蚁酸液、腐败土——调配、研磨、加热，试图合成出强大的炸药。

方法对了，坩埚冒出一股奇怪的烟，你得到一句模糊的提示：**"你好像搞出了什么东西……"**

是什么？不知道。你得自己去隔壁的试验场——点燃引线，炸一次，亲眼看看。

给它起个名字。然后带上它，去完成蚁后下达的任务。

```
   └──────────┘      └──────────┘      └──────────┘      └──────────┘
        ▲                                                      │
        └──────────────── 带回新材料，买新设备 ◀─────────────────┘
```

**合成 → 测试 → 命名 → 任务。** 每一个新炸药都是你亲手发现、亲手验证、亲手命名、亲手带出去用的。它不是配方书上的一个条目——它是属于你的一个小怪物。

## 实验室：树状合成图 + 设备解锁

合成不是"三个滑块盲调"——它是一张**可见的合成图**。

```
★ 黑火药 ─┬─ 栗色火药
           ├─ 慢燃火药
           ├─ ??? (需要操作: 极细研磨 Lv4)    ← 不告诉你买什么设备，自己去发现
           └─ ??? (需要操作: 液体蒸馏)
```

- **16 种天然原料**——从硫磺石到腐败土，每种原料在买了新设备后都可能暴露出隐藏的"???"分支
- **14 件可独立购买的设备**——不是一键升级实验室，而是**自己决定**先买蒸馏管还是先升研钵。每件设备解锁新操作能力
- **合成成功只给模糊提示**——"坩埚里冒出刺鼻的黄烟……"——你亲手去试验场引爆才知道它是什么
- **亲眼验证后，自己命名**——名字永久刻在合成图上

### 五条化学分支，覆盖全部经典爆炸物

| 分支 | 起点 | 路径 | 经典对应 |
|------|------|------|---------|
| 黑火药家族 | 硝石+硫磺+碳粉 | 栗色/慢燃/速燃火药 | Black Powder |
| 硝化纤维线 | 碳化纤维+硝化酸液 | 火棉→塑胶炸药 | Nitrocellulose→C4 |
| 芳香族线 | 天然树脂蒸馏 | 芳香油→TNT→苦味酸→特屈儿 | TNT→Picric Acid |
| 甘油线 | 肥脂+草木灰蒸馏 | 甘油→硝化甘油→达纳→爆胶 | Dynamite |
| 晶胺线 | 腐败土蒸馏发酵 | 蚁醛液→晶胺→RDX→HMX | RDX→HMX |

**硬核军迷会发现每一个经典化合物的影子。** 休闲玩家会凭好奇心撞出一条属于自己的路。

## 三个平行章节

### 第一章 · 泥土之下
蚁群在扩张。蚁后下令开凿新巢室。你带着炸药深入隧道——炸碎硬岩，打通通道。小心：炸塌了支撑柱，整个隧道都会坍塌。

> *"黑火药炸完只裂了条缝。也许我该试试那个黄色的大家伙……"*

### 第二章 · 蚁后的命令
地表有障碍——人类遗留的金属瓶盖、巨大的瓢虫尸体、蒲公英深根挡住了行军路线。蚁后命令清除。每个障碍的材质不同——纸板吃冲击力，金属需要穿甲爆破，植物从节点炸断——你需要为每个目标选对炸药。

> *"TNT 把纸板炸飞了，但那个铁瓶盖纹丝不动。我需要更狠的东西。"*

### 第三章 · 红火蚁入侵
红火蚁发动了全面进攻。前方侦察蚁报告：三个敌前哨巢穴，每个有土垒外墙、蚁酸罐、幼虫库。用实验室最强大的武器——熔岩爆弹、雷管起爆的 RDX、塑胶炸药贴墙炸——精确摧毁目标。蚁酸罐炸错了→酸雾飘回→自己人伤亡。

> *"这次不光是炸东西。这次是战争。"*

## 世界观：蚁族的微观宇宙

- 一片草叶 = 摩天大楼。一滴露水 = 湖泊。一颗锈铁钉 = 摩天楼。
- 你是行军蚁——工程师，不是吉祥物。手里攥的是真炸药。
- 暖黄萤火虫光 vs 深棕洞穴暗影——这个对比就是游戏的情绪。
- **铁律：不要可爱化。** 像素风不等于萌。这是认真工蚁在干危险的事。

## 为什么叫 AntChem

**Ant + Chem。** 两个音节。蚂蚁 + 化学工业（不是"炼金术"——是"工程"）。念出来顺口，Steam 上独一无二。

中文：**蚂蚁化工**。蚂蚁在地下挖洞、清除障碍、和别的蚂蚁打仗——这本来就是蚂蚁的天性。化工 = 设备 + 流程 + 产出物。你的 14 件设备从碎石研钵到高温熔炉，本身就是一条微型军工化学品生产线。
